# App i18n - Traducción Dinámica con Ionic y Angular

Esta es una aplicación móvil híbrida desarrollada con **Ionic** y **Angular**, diseñada para mostrar cómo implementar internacionalización (i18n) de manera local. Permite el cambio dinámico de idiomas en la interfaz de usuario en tiempo real.

## Tecnologías y Herramientas

* **Framework:** Angular 20
* **UI Components:** Ionic Framework
* **Librería de Traducción:** `@ngx-translate/core` & `@ngx-translate/http-loader`
* **Lenguaje:** TypeScript, HTML5, SCSS

## Características Principales

* Cambio dinámico entre idiomas (Español e Inglés) sin recargar la página.
* Carga de archivos de traducción mediante HTTP local (`.json`).
* Arquitectura limpia usando `TranslateService`.
